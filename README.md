# School Management System (Flask)

## Description
This repository contains a Flask-based School Management System that allows you to manage student records through a web interface.  
It demonstrates the use of Flask, PostgreSQL, and HTML templates to implement CRUD operations.

---

## Features
- User **Signup** and **Login** with password hashing.
- Add new students with details:
  - Roll number
  - Name
  - Fatherâ€™s name
  - Class
  - Subject
  - Grade
- View all students in a **paginated list**.
- Update student information.
- Delete students.
- Filter students by subject/class.
- Secure session management with login/logout.

---

## Tech Stack
- **Backend:** Python 3.x, Flask  
- **Database:** PostgreSQL  
- **Templating:** HTML, Jinja2  
- **Security:** Werkzeug for password hashing
